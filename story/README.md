# Story

This folder is a quarantined narrative archive. Only the Writer Agent reads or writes here.

No other agent — R1, R2, Strategist, Probability, Modeler, Lint, Browser Intern, main agent, or any Librarian (default or Writer-scoped, regardless of spawner) — may read, grep, reference, or write to `story/`. The quarantine is enforced in `program.md` (Rule 34 — STORY QUARANTINE) and in `soul_librarian.md` (Absolute prohibition: the `story/` folder).

Rationale: narrative is for the human reader only. If it loops back into research direction via Librarian grep or any other channel, it warps research toward the narrative framing, which warps next sessions' research, which warps next episodes — a feedback loop the quarantine prevents.
